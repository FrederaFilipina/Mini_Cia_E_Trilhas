Mini Cia && Trilhas

02 - Login do usuário


Objetivo:

Validar o acesso, permitindo que o usuário utilize dos recurso da plataforma. E alterar a senha em caso de esquecimento.

Processo de
Funcionamento:

    Login:
    • Entradas:
      • #01. Cadastro do usuário

    • Execução:
      1. Preencher os campos de Nome de Usuário e Senha
      2. Clicar no botão: Login
        • Se informações inseridas são verdadeiras, liberar acesso ao usuário
        • Se informações inseridas são falsas, alertar sobre erro

    • Saídas:
      • Acesso completo aos recursos da plataforma


    Troca de Senha:
    • Entradas:
      • #01. Cadastro do usuário

    • Execução:
      1. Clicar no botão para trocar a senha
      2. Preencher os campos
        • E-mail
        • CPF
      3. Digitar a nova senha
      4. Clicar em botão para salvar a nova senha

    • Saídas:
      • #01. Cadastro do usuário
