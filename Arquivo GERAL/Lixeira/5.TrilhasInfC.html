5. Trilhas Informações Completas


Objetivo:

  • Ter um local com todas as informações pertinentes para que
    os usuários possam visualizar e escolher qual trilha fazer

Processo de
Funcionamento:

  • Entradas:
    • Informações coletadas pelos administradores da plataforma

  • Execução:
    1. Display com as informações das trilhas
      • Definir layout de funcionamento da plataforma!
        • Vai ter um área de busca por nome
        • Vai ser subdividido por regiões?
      • Informações sobre a trilha:
        • Nome da Trilha (Ponto Inicial + Ponto Final)
        • Região que se encontra
        • CEP
        • Local de início
        • Local de chegada
        • Distância
        • Tempo estimado
        • Tipo de relevo
        • Elevação
        • Grau de dificuldade (definir ponstos por tempo, distância, relevo, elevação e fazer uma média)
        • Particularidades
    2. Clicar no botão para criar um evento
    3. Clicar no botão para participar de um evento

  • Saídas:
    • Trilhas cadastradas
    • Área na tela inicial com as informações básicas da trilha
    • Informações completas para selecionar na hora de criar o Evento
    • Informações completas para pesquisar as trilhas
    • Informações completas para visulizar nos Eventos disponíveis
    • Informações completas para avaliar a trilha

  • Dúvidas:
    • Como definir o grau de dificuldade?
