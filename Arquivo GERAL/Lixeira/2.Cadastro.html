2. Cadastro


Objetivo:

  • Registrar os dados do usuário em um "Banco de dados" de forma que posteriormente
    o mesmo possa acessar os recursos da plataforma por meio de login e senha

Processo de
Funcionamento:

  • Cadastro
    • Entradas:
      • As informações fornecidas pelo usuário

    • Execução:
      1. Clicar no botão de cadastro
        • Definir layout de funcionamento da plataforma!
      2. Prrencher os campos:
        • Nome completo
        • CPF (só os números)
        • Data e nascimento ( vai gerar a idade, formato: dd/mm/aaaa, utilizando um calendário ou digitando)
        • Sexo (feminino, masculino, outros abre uma opção de preenchimento)
        • Residente da Grande Florianópolis? (Ñ Obrigadorio!)
        • Se sim, Pedir o CEP (Ñ Obrigadorio!)
        • Tel.: Celular (Ñ Obrigadorio!)
        • Email
        • Senha (tamanho: mín.:04 e máx.:08 e Limitar alguns caracteres especiais)
        • Nome de usuário (tamanho mín.:04 e máx: 08 e Vai ser único eterá um aviso para nome de usuário já cadastrado.)
      3. Cliar no botão para finaliza o cadastro
      4. Fazer o login para acessar os reursos da plataforma

    • Saídas:
      • Cadastro feito realizado
      • Informações para constar nos eventos disponíveis
      • Informações para constar na avaliação dos participantes do evento

    • Dúvidas:
      • ....
