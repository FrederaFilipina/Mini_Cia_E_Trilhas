2. Cadastro


Objetivo:

  • Registrar os dados do usuário em um "Banco de dados" de forma que posteriormente
    o mesmo possa acessar os recursos da plataforma por meio de login e senha

Processo de
Funcionamento:

  • Cadastro
    • Entradas:
      • As informações fornecidas pelo usuário
    • Execução:
      1. Clicar na opção de cadastro
        • Vais er um botão ou um link de texto
      2. Vai abrir uma nova página com uma sessão de campos para preencher, ou esses campos vão aparecer logo abaixo?
      3. O usuário vai preencher os seguintes campos e quais serão brigatórios?
        • Nome completo
        • CFP (só os dígitos?)
        • Data de Nascimento(só os dígitos, qual o formato DD/MM/AA ou AAAA)
        • Sexo (M e F ou vamos colocar a família toda?)
        • Residente da Grande Florianópolis? (sim/não)
        • CEP
        • Tel.: Celular
        • E-mail
        • Senha (como vai ser? número, letras Maiusculas e Minusculas, caracteres especial, tamanho)
        • Nome de Usuário
      4. Após feito o cadastro, ele já vai ter acesso aos recursos ou vai precisar fazer login?
    • Saídas:
      • Cadastro feito realizado
