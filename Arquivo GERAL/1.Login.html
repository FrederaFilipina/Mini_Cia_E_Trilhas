Parte do projeto: 1. Login

Objetivo:

  • Criar uma área na tela inicial para que o usuário faça o login e tenha acesso as recursos da plataforma.

Funcionamento:

  • O usuário deve entrar com o DEFINIR (nome de usuário / e-mail / CPF) e Senha;

Recursos para funcionamento:

  • Acesso as informações do Cadastro;
  • Criação de um código para:
    • Validar as informações e conceder acesso;

Novas Ideias e Melhorias:

  1. Novas Ideias:
    • Após (x tentativas) DEFINIR o acesso será bloqueado e a solicitação de uma nova senha será solicitado;
  2. Melhorias:
    • Disponibilização de botão para redefinir a senha em caso de esquecimento;
